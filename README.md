# 📘 Nossa Primeira Wiki no Github
## 📚 Dicionário de Comandos Essenciais do Linux

Este repositório foi criado como parte da **Atividade Prática: Nossa Primeira Wiki no GitHub**, com o objetivo de construir uma Wiki colaborativa utilizando **Markdown** para documentar comandos essenciais do **Linux**.

O projeto reúne **30 comandos fundamentais**, amplamente utilizados no dia a dia de usuários e profissionais que trabalham com sistemas Linux, especialmente em ambientes acadêmicos e laboratorias. 

---
### 📂 Comandos essenciais do Linux
---

## 1. ls
Lista todos os arquivos do diretório

**Uso Prático:** Ver o que tem dentro de uma pasta antes de acessar ou modificar arquivos.

**Exemplo:**
```
Bash

is -la
```
## 2. cd
Navegar entre diretórios.

**Uso Prático:** Entrar em uma pasta específica para trabalhar nela.

**Exemplo:**
```
Bash

cd Documentos
```
## 3. pwd
Mostrar o caminho atual.

**Uso Prático:** Saber exatamente onde você está no sistema.

**Exemplo:**
```
Bash

pwd
```
## 4. touch
Criar arquivos vazios.

**Uso Prático:** Criar rapidamente arquivos de configuração ou scripts. 

**Exemplo:**
```
Bash

touch script.sh
```
## 5. nano
Editor de texto no terminal.

**Uso Prático:** Editar arquivos diretamente no terminal sem interface gráfica.

**Exemplo:**
```
Bash

nano arquivo.txt
```
## 6. cat
Exibir conteúdo de arquivos.

**Uso Prático:** Ver rapidamente o conteúdo de um arquivo pequeno.

**Exemplo:**
```
Bash

less arquivo.txt
```
## 7. less
Visualizar arquivos grandes.

Ler logs ou arquivos extensos com rolagem.

**Uso Prático:** Ler logs ou arquivos extensos com rolagem.

**Exemplo:**
```
Bash

less arquivo.txt
```
## 8. mkdir
Criar diretórios.

**Uso Prático:** Organizar arquivos em pastas.

**Exemplo:**
```
Bash

mkdir projetos
```
## 9. rm
Remover arquivos.

**Uso Prático:** Apagar arquivos desnecessários.

**Exemplo:**
```
Bash

rm arquivo.txt
```
## 10. rmdir
Remover diretórios vazios.

**Uso Prático:** Limpar estrutura de pastas.

**Exemplo:**
```
Bash

rmdir pasta
```
## 11. cp
Copiar arquivos.

**Uso Prático:** Fazer backup rápido de arquivos.

**Exemplo:**
```
Bash

cp arquivo.txt backup.txt
```
## 12. mv
Mover ou renomear arquivos.

**Uso Prático:** Organizar arquivos ou alterar nomes.

**Exemplo:**
```
Bash

mv arquivo.txt documentos/
```
## 13. find
Buscar arquivos.

**Uso Prático:** Encontrar arquivos perdidos no sistema.

**Exemplo:**
```
Bash

find . -name "arquivo.txt"
```
## 14. grep
Buscar texto dentro de arquivos.

**Uso Prático:** Procurar erros em logs ou palavras específicas.

**Exemplo:**
```
Bash

grep "erro" log.txt
```
## 15. chown
Alterar dono do arquivo.

**Uso Prático:** Ajustar permissões em servidores.

**Exemplo:**
```
Bash

chown usuario arquivo.txt
```
## 16. chmod
Alterar permissões.

**Uso prático:** Tornar scripts executáveis.

**Exemplo:**
```
bash

chmod +x script.sh
```
## 17. top
Monitorar processos em tempo real.

**Uso prático:** Ver programas que estão consumindo muita memória.

**Exemplo:**
```
bash

top
```
## 18. ps
Listar processos ativos.

**Uso prático:** Identificar processos em execução.

**Exemplo:**
```
bash

ps aux
```
## 19. kill
Encerrar processos.

**Uso prático:** Finalizar programas travados.

**Exemplo:**
```
bash

kill 1234
```
## 20. ping
Testar conexão de rede.

**Uso prático:** Verificar se um site está acessível.

*Exemplo:*
```
bash

ping google.com
```
## 21. wget
Baixar arquivos da internet.

**Uso prático:** Baixar arquivos direto do terminal.

*Exemplo:*
```
bash

wget https://exemplo.com/arquivo.zip
```
## 22. curl
Fazer requisições HTTP.

**Uso prático:** Testar APIs.

*Exemplo:*
```
bash

curl https://api.exemplo.com
```
## 23. df
Ver uso de disco.

**Uso prático:** Checar espaço disponível no sistema.

*Exemplo:*
```
bash

df -h
```
## 24. du
Ver tamanho de arquivos/pastas.

**Uso prático:** Descobrir o que está ocupando espaço.

*Exemplo:*
```
bash

du -h
```
## 25. clear
Limpar o terminal.

**Uso prático:** Organizar a visualização dos comandos.

*Exemplo:*
```
bash

clear
```
##26. history
Executar como administrador.

*Uso prático:* Instalar programas ou alterar sistema.

*Exemplo:*
```
bash

sudo apt update
```
##28. apt
Gerenciar pacotes.

**Uso prático:** Instalar programas.

*Exemplo:*
```
bash

sudo apt install git
```
## 29. uname
Informações do sistema.

**Uso prático:** Ver versão do sistema operacional.

*Exemplo:*
```
bash

uname -a
```
## 30. date
Mostrar data e hora.

**Uso prático:** Verificar horário do sistema.

*Exemplo:*
```
bash

date
```
